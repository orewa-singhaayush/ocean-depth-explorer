# 🌊 Ocean Depth Explorer

> “The Deeper You Go, The Less We Know.”

An immersive, scroll-driven storytelling website that visually explores the different layers of the ocean — from the bright surface to the mysterious depths.

---

## 🚀 Features

* 🌊 Scroll-based storytelling experience
* 🎬 Smooth animations using GSAP
* 🎯 Interactive navigation (header + dot navigation)
* 🐟 Fish jumping animation at the surface layer
* 📱 Fully responsive design

---

## 🌐 Sections Overview

The website is divided into multiple ocean layers:

* ☀️ **Surface** – Bright sky and ocean with animated fish
* 🌞 **Sunlight Zone** – Light penetration with visible marine life
* 🌅 **Twilight Zone** – Fading light and mysterious environment
* 🌌 **Midnight Zone** – Complete darkness with glowing creatures
* 🕳️ **Abyssal Zone** – Extreme depth and pressure
* ⚫ **Trench Zone** – Deepest parts of the ocean


---

## 🎯 Concept

This project simulates a deep-sea dive using scroll-based interactions.
As the user scrolls down, each section represents a deeper layer of the ocean, with changing visuals, colors, and animations to create an immersive experience.

---

## 🛠️ Technologies Used

* HTML
* CSS
